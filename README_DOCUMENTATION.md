## 📱 Device Testing Quick Reference

From **RESPONSIVE_DESIGN_REFERENCE.md**:

### Common Screen Sizes
- **Mobile**: 320px - 480px
- **Tablet**: 600px - 1024px
- **Desktop**: 1024px+

### How to Test
1. **DevTools**: Press F12 → Ctrl+Shift+M
2. **Real Device**: http://[your-ip]:3001
3. **Responsive**: Resize browser window
